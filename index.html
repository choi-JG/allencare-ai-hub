HOW-TO

노하우 팁

AI를 더 잘 활용하기 위한 실용 가이드와 프롬프트 작성법입니다.

"10년 경력의 마케터처럼 답해줘"처럼 역할을 먼저 정의하면 훨씬 전문적이고 상황에 맞는 답변이 나옵니다. AI는 역할에 따라 사용하는 어휘, 깊이, 관점이 달라집니다.

# 예시 프롬프트
당신은 B2B SaaS 마케팅 10년 경력의 전문가입니다.
우리 제품의 LinkedIn 광고 문구 3가지를 작성해주세요.

"3개 항목으로", "표 형식으로", "300자 이내로"처럼 형식을 명시하면 바로 복사해서 쓸 수 있는 결과물이 나옵니다. 형식 지정 없이 요청하면 불필요하게 길어질 수 있습니다.

# 형식 지정 예시
아래 내용을 불릿 포인트 5개로, 각 항목 50자 이내로 요약해줘.
표 형식으로 정리하되 열은 항목/설명/우선순위로 구성해줘.

"이메일 써줘"보다 "B2B 고객 대상, 계약 갱신 독려 목적, 정중하고 간결하게, 200자 내외"처럼 상황을 설명할수록 정확한 결과가 나옵니다. 맥락이 많을수록 AI는 더 정확해집니다.

# 좋은 프롬프트 = 역할 + 상황 + 과제 + 형식
[역할] 10년 경력 B2B 영업 전문가
[상황] HR SaaS 솔루션 판매, 중소기업 대상
[과제] 신규 리드 첫 콜드 이메일 작성
[형식] 제목 1줄 + 본문 150자 + CTA 1개, 3가지 버전

첫 답변에 만족하지 말고 이어서 수정 요청하세요. "더 간결하게", "공격적인 마케팅 톤으로", "전문 용어 빼고"처럼 피드백하면 점점 원하는 결과가 됩니다.

# 이어서 수정하는 방법
→ "더 짧게 줄여줘" (절반 길이로)
→ "2번 버전을 더 부드러운 톤으로 바꿔줘"
→ "수치를 더 강조하고 감정적 표현 줄여줘"

AI는 틀린 정보를 자신 있게 말하기도 합니다 (할루시네이션). 수치, 통계, 법률 정보, 인용문은 반드시 원본 출처를 확인하세요. AI는 초안 도구이지 최종 검증 도구가 아닙니다.
